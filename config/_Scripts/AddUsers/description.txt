#NoEnv

; 1. Создаётся пользователь со случайным паролем
; 2. Пароль отправляется через форму
; исходные данные в формате: пользователь - URL формы - шаблон блока POST (куда будут подставляться hostname, username и пароль)

; 3. Пароль также записывается в случайный файл на компьютере пользователя в папке, из которой нельзя получить список файлов.

; 4. Скрипт на компьюете пользователя может периодически проверять этот файл и менять пароль на настраиваемом компьютере
